Jonathan Wheelock Do you need a thing? Make it. Did you successfully make it?

Do you need a program? Program it. Did you successfully program it?

Do you need to get from point A to point B using a car? Drive it. Did you make it there in one piece?

Obviously your success rate at any of these endeavors will likely never be 100%, so a failure doesn't disqualify you, but does require more testing. These activities are sufficiently complex such that success by dumb luck doesn't seem a plausible outcome, limiting the concern for false positives.
